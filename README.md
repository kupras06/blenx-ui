# Blenx UI

Build modern React applications with accessible, customizable, registry-first components powered by StyleX and Base UI.

Blenx UI brings the developer experience of shadcn/ui to the StyleX ecosystem. Instead of installing a large component library, add only the components you need and own the source code from day one.

## Why Blenx?

Most component libraries force you to choose between flexibility and speed.

Blenx is built around a different philosophy:

- Own your components
- Copy source code into your project
- Customize everything
- No vendor lock-in
- Accessibility built-in
- StyleX-first architecture

## Philosophy

Blenx follows a registry-first approach.

Instead of consuming components from a black-box package, components are added directly to your application.

This allows teams to:

- Modify implementations freely
- Establish internal design standards
- Scale design systems across multiple products
- Avoid dependency lock-in
